Задание.

Пользователь с консоли вводит целое число(1-10 включительно), которое интерпретируется как размер массива(двухмерный массив N на N)
Эту квадратную матрицу заполняем целыми числами рандомом(от -100 до 100 включительно)

1. Выводим матрицу в файл (при работе с файлами использовать Exception)
2. Выводим матрицу в файл спиралью (спираль закручивается во внутрь начиная с левого верхнего символа, вывод строкой)
3. Выводим матрицу в файл змейкой (туда-назад, старт змейки с правого нижнего символа, вывод строкой)

All done.
